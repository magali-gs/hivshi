# HivsHi: Experimental JavaScript Project

HivsHi is a creative and experimental JavaScript project developed as part of the SuperHi Experimental JavaScript course. It's an interactive web experience featuring physics-driven shapes and real-time interactions using the Matter.js physics engine.
